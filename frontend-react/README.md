# Frontend React - RECETAS-FINALCURSO

Cliente React (Vite) que consume la API del backend en `http://localhost:3000/api/v1/recetas`.

## Funcionalidades incluidas

- Listado con paginación (`/paginate`) y filtro por categoría.
- Búsqueda local por nombre y descripción.
- CRUD completo: crear, editar, ver detalle y eliminar.
- Formularios controlados con validaciones funcionales.
- `useState` y `useEffect` para manejo de estado/ciclo de vida.
- React Router para navegación (`/`, `/receta/:id`, `/nueva`, `/editar/:id`).
- Bootstrap para estilos base y alertas de éxito/error.

## Comandos

```bash
npm install
npm run dev
```

Acceso por defecto: `http://localhost:5173`
